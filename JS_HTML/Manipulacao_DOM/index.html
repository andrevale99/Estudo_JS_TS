Acesso de elementos DOM

Há comandos de acesso aos elementos HTML utilizando o javascript, alguns serão mostrados a seguir.

  1. document.getElementById():

    Maneira de acessar algum elemento a partir de sua id, digitando a linha abaixo no console co navegador irá retorna o objeto deste parágrafo (tudo). Já o segundo irá retornar este texto.

    document.getElementById('exemplo_1') document.getElementById('exemplo_1').textContent
  2. document.getElementsByClassName():

    Irá retornar o mesmo exemplo anterior, porém de vez utilizar a ID, deve-se utilizar a CLASS da tag. O exemplo irá retornar todos os elementos que utiliza da classe CSS de "codes".

    document.getElementsByClassName('codes')
