Acesso de elementos DOM

Há comandos de acesso aos elementos HTML utilizando o javascript, alguns serão mostrados a seguir.

  1. document.getElementById():

    Maneira de acessar algum elemento a partir de sua id, digitando a linha abaixo no console co navegador irá retorna o objeto deste parágrafo (tudo). Já o segundo irá retornar este texto.

    document.getElementById('exemplo_1') document.getElementById('exemplo_1').textContent

  2. document.getElementsByClassName():

    Irá retornar o mesmo exemplo anterior, porém de vez utilizar a ID, deve-se utilizar a CLASS da tag. O exemplo irá retornar todos os elementos que utiliza da classe CSS de "codes".

    document.getElementsByClassName('codes')

  3. document.getElementsByTagName():

    Irá retornar os objetos que possui a tag HTML selecionada na função. O exemplo irá retornar todos os parágrafos desta página HTML.

    document.getElementsByTagName('p')

Usando o querySelector

Deve-se colocar dentro das duas funções a seguir os seletores desejados. Os seletores são: classe CSS (.nome_da_classe_css), # (ID do elemento), e outros que esqueci.

  1. document.querySelector():

    Pode-se também utilizar a função para retornar o primeiro elemento da página.

  2. document.querySelectorAll():

    Também pode utilizar o par retornar todos os elementos da página.

Alguns comandos de navegaçao

podemos acessar os elementos filhos de um elemento pai a partir de algumas propriedades, dentre elas:

  1. document.querySelector('p').parentNode:

    Retorna o elemento pai do seletor.

  2. document.querySelector('body').childNodes:

    Retorna a lista com todos os nós filhos do seletor. Também retornar alguns Nós de texto que são relacionados a formatação, tabulação, dentre outros.

  3. document.querySelector('body').children:

    Caso não queira retornar os Nós de texto, utiliza-se essa função que retorna somente os elementos HTML (div, p, h2, dentre outros).

    4. Também a outras propriedades: "nextSibling e previousSibling, que representam, respectivamente, o próximo e o anterior irmão do nó em questão"; " Caso queiramos filtrar somente elementos HTML, usamos as propriedades nextElementSibling e previousElementSibling ." (IMD, Aula 05 - Manipulação do DOM)

    const teste = document.querySelector('#item').children[1];
    // Acessando o segundo filho do elemento de id 'item' teste.nextElementSibling.style.background = 'orange';
    // Mudando a cor de background do próximo irmão do elemento de id 'item'
